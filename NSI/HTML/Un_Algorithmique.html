Un algorithme : Le PageRank


En 1966, Larry Page et Sergey Brin , créent le PageRank. C'est sans aucun doute l’algorithme le plus utilisé dans le monde.

Définition

Le PageRank est un algorithme utilisé par Google. Il permet de mesurer la popularité d’un site ou d’une page web. C’est un système qui a été inventé par Larry Page, qui n’est autre que le co-fondateur de Google.

Son utilité

Il s'agit du système permettant le classement des pages Web utilisé par le moteur de recherche Google. Il mesure quantitativement la popularité d'une page web.
Le PageRank n'est qu'un indicateur parmi d'autres dans l'algorithme qui permet de classer les pages du Web dans les résultats de recherche de Google.

Son fonctionnement

Il est représenter sur une echelle de valeur allant de 0 à 10.
L’échelle du PageRank n’est pas linéaire : ainsi, il est bien plus difficile de passer d’une valeur de PageRank de 3 à 4 que de 2 à 3. La popularité est un facteur important permettant d’établir le classement des pages web dans les résultats du moteur de recherche. Elle s’acquiert principalement par la quantité et la popularité des liens pointant sur une page.
Par ailleur, le pagerank étudie également la qualité des sites qui en parlent et/ou qui utilisent ces liens sortant. Ainsi, plus les gens qui parlent d’un site, plus le PageRank sera élevé.
Et si vous désirez connaître votre indice ParegRank, alors il existe plusieurs outils qui permettent de le calculer en ligne, le plus connu étant « Google Bar ».
Pour conclure, un lien provenant d’une page dont le PageRank est de 4 est plus qualitatif qu’un lien provenant d’une page dont le PageRank est de 3.
